# FIBONO

Projeto de site simples para funcionar como cartão de visita online.

## Como usar

### Rodar localmente

1. Abra o arquivo `index.html` no navegador.

### Deploy na Vercel

1. Conecte este repositório no Vercel (https://vercel.com).
2. Escolha o time/projeto e selecione o repositório.
3. A Vercel detecta automaticamente um site estático e fará o deploy.

## Estrutura inicial

- `index.html` — página principal do cartão de visita.
- `styles.css` — estilos básicos.
- `vercel.json` — configuração de deploy para Vercel.
